# tic_tac_toe
Game
Tic Tac Toe is a classic two-player strategy game played on a 3×3 grid. Each player takes turns marking a box with their symbol:

- Player 1: ❌ (X)

- Player 2: ⭕ (O)

The goal is to be the first player to get three of your symbols in a row — horizontally, vertically, or diagonally.
